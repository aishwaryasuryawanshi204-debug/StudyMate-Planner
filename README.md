\# StudyMate - Daily Planner



\## 📌 Description

StudyMate is a simple Java-based application to manage daily study tasks.



\## ✨ Features

\- Add tasks

\- View tasks

\- Mark tasks as completed

\- Delete tasks



\## 🛠️ Technologies

\- Java

\- OOP Concepts



\## 🚀 How to Run

1\. Compile: javac Main.java

2\. Run: java Main
